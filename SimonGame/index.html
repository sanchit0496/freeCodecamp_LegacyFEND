How To Play This Game

  • The user is presented with a random series of button presses.
  • Each time the user input a series of button presses correctly, he sees the same series of button presses but with an additional step.
  • The user hears a sound that corresponds to each button both when the series of button presses plays, and when the user personally presses that button.
  • If the user presses the wrong button, he is notified that he has done so, and that series of button presses starts again to remind him of the pattern so he can try again.
